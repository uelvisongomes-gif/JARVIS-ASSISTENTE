# JARVIS ASSISTENTE

Extensão inteligente para ajudar programação, automações e análise de telas em tempo real.

## Objetivo

Criar uma extensão do Chrome com:

- Bolinha flutuante
- Painel lateral
- Captura de tela
- Análise da página atual
- Comunicação com IA
- Assistente para GitHub, Lovable, Supabase, Vercel e programação

---

## Funcionalidades

### MVP
- Abrir painel lateral
- Chat com IA
- Capturar contexto da página
- Enviar print da tela
- Receber instruções em tempo real

### Futuro
- Sugestões automáticas
- Leitura de erros
- Correção de código
- Integração com GitHub
- Integração com Lovable
- Agente autônomo
